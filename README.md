# Projet Site InovTech

**Projet réalisé dans le cadre de la formation DSP DevOps au CNAM**.  
Ce projet a été un **travail de groupe de 3 personnes** :  
- **Aïcha FOFANA**  
- **Meriem MEZDARI**  
- **Codou diouf FAYE**  

## Contexte du projet
Dans le cadre de notre formation, nous devions créer un site internet de notre choix.  
Nous avons imaginé et conçu un site pour des **stylos traducteurs**, sous le nom de **InovTech**.  

Le projet nous a permis de :  
- Créer plusieurs pages HTML/CSS multi-pages  
- Définir une **charte graphique commune**  
- Travailler en équipe et répartir les responsabilités  
- Appliquer les bonnes pratiques de structuration de fichiers

## Répartition du travail
| Page / Fichier              | Responsable |
|-----------------------------|-------------|
| `inovtech.html` + `inovtech.css`         | Meriem MEZDARI |
| `formulaire.html` + `formulaire.css`     | Codou diouf FAYE |
| `page-produit.html` + `page-produit.css` | Aïcha FOFANA |
| `panier.html` + `panier.css`             | Les 3 membres |

La **conception des pages** et la **charte graphique** ont été réalisées **ensemble par les 3 membres du groupe**.
